# Security Policy

## Supported versions

Security fixes are applied to the latest released version of Alfa API SDK.

## Reporting a vulnerability

Please do not disclose security vulnerabilities publicly.

Report suspected vulnerabilities through a private GitHub security advisory for this repository, or contact the maintainers directly if private advisories are not available.

Include:

- Affected module and version.
- Minimal reproduction details.
- Expected and actual impact.
- Any known workaround.
